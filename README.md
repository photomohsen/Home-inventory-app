# Mohsen's Home Assistant Add-ons

Add-on repository for Home Assistant.

## Installation

[![Add repository to my Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fphotomohsen%2FHome-inventory-app)

Or manually: **Settings → Add-ons → Add-on store → ⋮ → Repositories** and add:

```
https://github.com/photomohsen/Home-inventory-app
```

## Add-ons

### [Home Inventory](./home_inventory/)

Find anything in your home. Search-first household inventory with visual storage maps
(rooms → storage units → compartments), trilingual EN/فارسی/Dansk, barcode & QR scanning,
photos, borrow tracking, and NFC check-in/check-out via Home Assistant tags.
